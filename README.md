# GéoWatch — Observatoire des conflits

Plateforme de veille géopolitique et géostratégique avec focus Burkina Faso.

## Fonctionnalités

- 11 fiches conflits méthodologie 8 dimensions (gabarit Note de Situation CNES)
- Analyses simplifiées + briefs 2 couches (décideur + analyste)
- Scénarios prospectifs (méthode Godet)
- Indicateurs à surveiller (24-72h, 7-30j)
- Carte stratégique mondiale en français (capitales, détroits, ports, bases, corridors)
- Page « Impacts Burkina Faso » dédiée
- Veille RSS (11 sources vérifiées + auto-refresh + notifications navigateur)
- Exports PDF, Word, PPTX, TXT

## Démarrage

Ouvrir `index.html` dans un navigateur moderne (Chrome, Edge, Firefox).

Aucune installation, 100% client-side, données en localStorage.
