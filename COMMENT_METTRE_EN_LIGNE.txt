=========================================================
  GÉOWATCH — MISE EN LIGNE (méthodologie + 2 agents autonomes)
=========================================================

CETTE MISE À JOUR AJOUTE :
  • 5 pages : Matrice de pertinence · Desks/Répartition · Méthodologie ·
    Cadence/Livrables · Études & Analyses
  • Agent de collecte « méthodologie-aware » (étiquette zone/thématique,
    matrice, routage par desk, brief par desk).
  • Agent ANALYSTE (nouveau) : détecte tendances lourdes, signaux faibles,
    ruptures, recompositions/bascules ; produit notes d'analyse, prospectives
    et études thématiques ; alimente la page « Études & Analyses ».

MÉTHODE A — github.com (glisser-déposer en conservant les dossiers)
-----------------------------------------------------------------
1. Dézippe ce paquet : tu obtiens l'arborescence exacte (methodo.js, index.html,
   app.js, et les dossiers scripts/, .github/, data/).
2. Ouvre  https://github.com/sawadogorachad-coder/geowatch
3. « Add file » → « Upload files ».
4. Glisse TOUT le contenu dézippé d'un coup (GitHub conserve les sous-dossiers).
5. Message de commit : « Méthodologie + agents (collecte & analyste) »
6. « Commit changes ».
7. Onglet « Actions » : déploiement + passage des agents (~2-3 min).
8. Ouvre  https://sawadogorachad-coder.github.io/geowatch/  puis Ctrl + F5.

OPTION IA (rédaction d'analyses en prose) — facultatif
-----------------------------------------------------------------
Par défaut, l'agent analyste fonctionne SANS IA (synthèses structurées, gratuit).
Pour activer la rédaction d'analyses par l'IA :
  1. Crée une clé API sur  https://console.anthropic.com  (quelques centimes/usage).
  2. Dans GitHub : Settings → Secrets and variables → Actions → New secret
     Nom :  ANTHROPIC_API_KEY      Valeur : ta clé (sk-ant-...)
  (Optionnel) Secret  GEOWATCH_LLM_MODEL  pour changer de modèle
     (ex. claude-haiku-4-5 pour réduire le coût).
L'agent détecte la clé tout seul et se met à rédiger ; sans clé, il reste gratuit.
